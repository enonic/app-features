Push Notifications Lib Demo

Exercises /lib/notifications (lib-notifications 3.0.0). The browser subscribes to the push service using the server's VAPID public key; the server then sends push notifications with send(), sendAsync() or against all stored subscriptions via the same controller. The same store is exposed headlessly via the universal API /api/com.enonic.app.features:notifications.
