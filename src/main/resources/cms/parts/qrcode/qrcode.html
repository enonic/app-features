QR Code Lib Demo

Library: enonic/lib-qrcode — encodes any payload as a PNG QR Code via qrcode.generateQrCode({text, size}). The library's only public function returns a binary PNG stream, so this demo shows the cases the API actually supports: different payload types, different image sizes, and the two ways to deliver the PNG to a browser (inline data URI vs. raw image/png response from a controller route).

API limitations vs. classic QR libraries: lib-qrcode 3.0.0 does not expose error-correction levels (L/M/Q/H), output format selection (SVG), margin/quiet-zone, or scale independently of size. Those parameters are not configurable; the lib uses sensible QRGen defaults.
