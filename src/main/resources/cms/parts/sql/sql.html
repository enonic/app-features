SQL Lib Demo

This part exercises lib-sql against an embedded H2 in-memory database. On every request the part runs ten cases end-to-end: DDL, INSERT seed, SELECT, queryFirst, parameter binding, UPDATE, DELETE, transaction commit, transaction rollback, error handling. The pooled connection handle is cached across requests; the schema is dropped and recreated on each request so the results are deterministic.
