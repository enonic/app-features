Markdown Lib Demo

A fixed set of demo cases covering the features and edge cases of com.enonic.lib:lib-markdown.

Use cases

Each case shows a Markdown input and the HTML produced by markdown.render(input). The lib-markdown 2.0.0 API exposes a single render(string): string function with no configuration options, so behavior here is exactly what plain CommonMark-Java produces.
